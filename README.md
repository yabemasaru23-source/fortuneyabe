# 四柱推命鑑定サイト

## 概要

生年月日と時刻を入力するだけで、伝統的な四柱推命の命式を自動生成し、大運（10年周期の運勢）まで表示するWebアプリケーションです。
添付の大森さんの命式PDFのようなフォーマットで結果を表示します。

## 主な機能

### ✅ 現在実装済みの機能

1. **基本情報入力フォーム**
   - お名前入力
   - 生年月日入力（カレンダー選択）
   - 生まれの時刻入力
   - 性別選択（男性/女性）
   - 出生地入力

2. **四柱推命計算**
   - 年柱、月柱、日柱、時柱の自動計算
   - 天干（10干）と地支（12支）の組み合わせ
   - 蔵干（地支に含まれる天干）の表示
   - 通変星（10神）の計算と表示
   - 十二運星の計算と表示

3. **大運計算**
   - 10年ごとの運勢表示
   - 大運の干支、通変星、十二運星
   - 大運天中殺の表示

4. **特殊情報**
   - 天中殺の有無チェック
   - 生月中殺の表示

5. **和風デザイン**
   - 伝統的な日本風の配色
   - レスポンシブデザイン対応
   - スマートフォン・タブレット対応

## 使用方法

1. **基本情報入力**
   - トップページのフォームに必要事項を入力
   - 生年月日はカレンダーから選択可能
   - 時刻は24時間形式で入力

2. **鑑定実行**
   - 「鑑定開始」ボタンをクリック
   - 数秒で結果が表示されます

3. **結果確認**
   - 四柱の命式表で各柱の詳細を確認
   - 大運表で10年ごとの運勢を確認
   - 特殊情報（天中殺など）を確認

4. **再鑑定**
   - 「再鑑定する」ボタンで新しい鑑定を開始

## 四柱推命の計算内容

### 年柱
- 対象年の干支を計算
- 1900年を基準に60進法で計算

### 月柱
- 年柱の天干から月の天干を決定
- 月の地支は固定（寅月から開始）

### 日柱
- 2000年1月1日（戊辰）を基準に日数計算
- 60進法で循環

### 時柱
- 日柱の天干から時の天干を決定
- 時の地支は2時間ごとに変化

### 通変星
- 日柱の天干を基準に他の柱との関係を判定
- 五行の相生相克関係から10神を決定

### 大運
- 月柱を基準に順送りまたは逆送りで計算
- 性別と月柱の天干の陰陽で方向を決定

## 注意事項

- このアプリケーションは簡易的な計算を行うため、専門的な四柱推命ソフトとの結果に差異が生じる場合があります
- 節入り日の考慮は簡易的な実装となっています
- 正確な鑑定を希望される場合は、専門家にご相談ください
